# 📰 Crypto News Bot

Telegram-бот для получения актуальных криптовалютных новостей, цен и топ движений рынка в реальном времени.

## 🚀 Возможности

- 📰 **Крипто-дайджест** — свежие новости из мира криптовалют, обработанные AI
- 💰 **Цены** — актуальные курсы BTC, ETH, SOL, BNB
- 🔥 **Топ движений** — топ-5 монет по изменению цены за 24 часа

## 🛠 Технологии

| Компонент | Технология |
|---|---|
| Бот | Python + aiogram 3.x |
| Автоматизация | n8n |
| Новости | RSS + Groq AI |
| Цены | CoinGecko API |
| Деплой | Railway |

## 📁 Структура проекта

    crypto-news-bot/
    ├── bot.py              # Основной код бота
    └── requirements.txt    # Зависимости

## ⚙️ Установка и запуск

**1. Клонируй репозиторий**

    git clone https://github.com/AKO71100/Crypto-News-Bot.git
    cd Crypto-News-Bot

**2. Установи зависимости**

    pip install -r requirements.txt

**3. Создай бота в Telegram**

- Открой @BotFather
- Напиши /newbot и следуй инструкциям
- Получи токен

**4. Настрой переменную окружения**

    export BOT_TOKEN=ваш_токен_от_botfather

**5. Запусти бота**

    python bot.py

## ☁️ Деплой на Railway

1. Форкни репозиторий
2. Зайди на railway.app
3. Создай новый сервис → выбери GitHub репозиторий
4. Добавь переменную BOT_TOKEN во вкладке Variables
5. Railway автоматически задеплоит бота

## 🔄 Как работает

    Пользователь нажимает кнопку
            ↓
    aiogram получает сообщение
            ↓
    Запрос к n8n webhook
            ↓
    n8n обрабатывает данные (RSS / CoinGecko / Groq AI)
            ↓
    Готовый текст возвращается боту
            ↓
    Бот отправляет ответ пользователю
